# House Price Prediction

## Project Overview

This project aims to analyze housing data and build a machine learning model to predict house prices based on property features.

The goal is to understand which factors influence house prices and to create a regression model that can estimate prices accurately.

---

## Dataset Description

This project uses the King County House Sales dataset, which contains house sale prices for homes in King County, including Seattle.

The dataset includes properties sold between May 2014 and May 2015, along with features such as number of bedrooms, bathrooms, living area, condition, and location.

It is commonly used for building and evaluating regression models.

**Source:** Kaggle (via Coursera)

## Exploratory Data Analysis

Exploratory data analysis was performed to understand relationships between features and house prices. Various visualizations were used to identify patterns and trends in the data.

---

## Model evaluation

- A Linear Regression model was used to predict house prices based on selected features.
- The model was trained on the dataset and used to estimate house prices.

---

## Key Insights

- Living area is one of the most important factors affecting price  
- Houses with more bedrooms and bathrooms generally have higher prices  
- There is a strong positive relationship between size and price  
- Some outliers exist with unusually high prices
